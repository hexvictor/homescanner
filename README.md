# HomeScanner

HomeScanner is a Next.js 14 application that provides an intuitive platform for home scanning.

## Technologies Used

- Next.js 14
- React
- Typescript
- Tailwind CSS
- MongoDB
- Mongoose
- NextAuth
- React Icons
- Photoswipe
- Cloudinary
- Mapbox
- React Map GL
- React Geocode
- React Spinners
- React Toastify
- React Share
